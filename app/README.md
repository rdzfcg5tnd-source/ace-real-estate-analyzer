# ACE 不動産投資分析アプリ

レインズで取得した販売図面PDFを見ながら項目を入力し、利回り・IRR・ROIを算出する投資分析ツール。
完全ローカル動作（サーバー・APIキー不要）。データはブラウザのlocalStorageに保存されます。

## 起動

```bash
cd app
npm install   # 初回のみ
npm run dev
```

ブラウザで http://localhost:5173 を開く。

## 公開（デプロイ）

サーバー不要の静的サイトなので、`npm run build` で生成される `dist/` を静的ホスティングに載せるだけで公開できます。
**いずれのホストも「プロジェクトのルートディレクトリ＝この `app/` フォルダ」**に設定してください。

### Vercel（推奨）
1. このリポジトリをGitHubに push
2. Vercel で New Project → リポジトリを選択
3. **Root Directory = `app`** を指定（Framework: Vite は自動検出）
4. Deploy（`app/vercel.json` の設定が自動適用）

### Netlify
- New site → リポジトリ選択 → **Base directory = `app`**（`app/netlify.toml` を自動適用）

### Cloudflare Pages
- ビルドコマンド `npm run build` ／ 出力ディレクトリ `dist` ／ ルート `app`

### ローカルで本番ビルドを確認
```bash
npm run build      # dist/ を生成
npm run preview    # 本番ビルドを http://localhost:4173 で確認
```

> 補足: 独自ドメインを使う場合は各ホストのダッシュボードで設定し、`index.html` の
> `<link rel="canonical">` と OGP の `og:image` URL を本番ドメインに更新してください。
> 画像認識（OCR）の日本語データは初回のみCDNから取得します（オンライン必須）。

## アプリ化（PWA / ストア配信）

- **PWA**: `vite-plugin-pwa` を導入して manifest とサービスワーカーを追加すれば、
  「ホーム画面に追加」でインストール可能・オフライン起動に対応（既存コードのまま）。
- **ストア配信**: Capacitor で iOS/Android のネイティブシェルに包む（コードはほぼ流用）。
  デスクトップは Tauri。

## 使い方

1. 左上「PDFを選択」で販売図面PDFを開く（図面を見ながら入力できます）
2. 右側に **価格・家賃・管理費・修繕積立金・面積・築年** を入力
3. 「投資前提」でローン条件・保有年数・売却想定などを調整（既定値入り）
4. 下部に **表面利回り / 実質利回り / IRR / ROI** とキャッシュフロー推移が即時表示
5. 「＋新規分析」で複数物件を保存・比較

## 主な機能

- **PDFプレビュー＋手入力** … 販売図面を見ながら項目入力
- **利回り・IRR・ROI分析**（税引前/税引後）
- **税引後CF・減価償却の精緻化** … 中古耐用年数・建物割合・所得税・譲渡所得税
- **物件の並列比較ビュー**（「比較」タブ）… 各指標の最良値を★でハイライト
- **Excel / PDFレポート出力**（分析結果ヘッダーのボタン）
- localStorageに自動保存

## 計算ロジック（src/lib/finance.ts）

| 指標 | 定義 |
|---|---|
| 表面利回り | 年間家賃 ÷ 価格（満室想定）|
| 実質利回り | (年間家賃 − 年間運営費) ÷ 総取得費 |
| ROI（税引前/税引後）| 初年度CF ÷ 自己資金（Cash on Cash）|
| IRR（税引前/税引後）| 保有期間の全CF（空室・家賃下落・ローン返済・減価償却・所得税・売却損益）の内部収益率 |

- 年間運営費 = (管理費+修繕積立金)×12 + 固定資産税 + 管理委託費 + その他
- 固定資産税が未入力なら概算（価格×0.6×1.7%）を使用
- ローンは元利均等返済（支払利息／元金を年次分離）
- **減価償却**: 建物価格(=価格×建物割合) ÷ 中古耐用年数（簡便法）
  - 中古耐用年数 = 経過≥法定: 法定×0.2 ／ 経過<法定: (法定−経過)+経過×0.2
  - 法定耐用年数: RC/SRC 47年・重量鉄骨34年・軽量鉄骨27年・木造22年
- **所得税** = 不動産所得（NOI−支払利息−減価償却）× 実効税率（赤字は損益通算による節税と仮定）
- **譲渡所得税** = (売却価格−簿価−売却費)× 税率（長期20.315%／短期39.63%）

## 出力

- **Excel**: 「サマリー」「キャッシュフロー」2シートの .xlsx（SheetJS）
- **PDF**: 印刷ダイアログ経由でレポートを出力（日本語フォント対応。ポップアップ許可が必要）
